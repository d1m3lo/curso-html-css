Você pode usar a estrutura style fora da estrutura body + head, para parametrar algumas coisas como modificar a posição das palavras(meio, direita, esqueda, etc..) e a cor das mesmas

<div> é usada para agrupar elementos(como textos, imagens, vídeosm links)em blocos, para que você possa organizar, estilizar ou controlar esses elementos como um conjunto.
